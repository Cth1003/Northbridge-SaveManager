Northbridge Save Manager Subsystem.

This tool allows the developers to implement user friendly features (import, export, backup and restore save files) in a seamless and easy way. While the tool was designed for RPG Maker VX Ace games, developers can take the source code and adjust it so it can work for their engine of their choice. Covered under GPLv2 license.
